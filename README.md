# 🎬 Nanobanana Pro - 3×3 シーンカット生成ツール

1枚の写真から映画のような9種類のカメラアングルを自動生成するツールです。

## ✨ 特徴

- 📸 1枚の写真から3×3グリッドで9種類のカットを生成
- 🎥 映画的なカメラアングル（超広角、ロング、クローズアップなど）
- 🎨 Google Nanobanana Pro (Gemini 3 Pro Image) 使用
- 💾 好きなカットを選んでZIPダウンロード
- ⬆️ オプションで高画質化（2倍解像度）

## 🚀 使い方

### オンライン版（推奨）

1. [こちらのリンク]にアクセス
2. fal.ai APIキーを入力（下記参照）
3. 画像をアップロード
4. 生成開始！

### fal.ai APIキーの取得方法

1. [fal.ai](https://fal.ai) でアカウント作成（無料）
2. ダッシュボードで API Key を取得
3. アプリに入力して使用

**料金**: 1K解像度 $0.15/回、4K解像度 $0.30/回  
**無料枠**: 新規アカウントで無料クレジットあり

## 💻 ローカルで動かす場合

```bash
# 1. リポジトリをクローン
git clone [YOUR_REPO_URL]
cd nanobanana-scene-cutter

# 2. 必要なパッケージをインストール
pip install -r requirements.txt

# 3. アプリを起動
streamlit run app.py
```

ブラウザで `http://localhost:8501` が開きます。

## 📋 生成されるカット一覧

1. **超広角ショット (ELS)** - 環境全体を捉える
2. **ロングショット (LS)** - 全身が見える
3. **ミディアムロング (3/4)** - 膝上から
4. **ミディアムショット (MS)** - 腰から上
5. **ミディアムクローズアップ (MCU)** - 胸から上
6. **クローズアップ (CU)** - 顔のアップ
7. **超クローズアップ (ECU)** - 目や手などのディテール
8. **ローアングル** - 下から見上げる
9. **ハイアングル** - 上から見下ろす
