## 🚀 友達に渡すための3ステップ

### ステップ1: GitHubにアップロード
1. https://github.com でアカウント作成（持ってなければ）
2. 「New repository」で新しいリポジトリを作成
3. ZIPの中身のファイルを全部アップロード

### ステップ2: Streamlit Cloud でデプロイ
1. https://streamlit.io/cloud でサインイン（GitHubアカウント使用）
2. 「New app」をクリック
3. さっき作ったリポジトリを選択
4. 「Deploy!」をクリック

### ステップ3: URLを友達にシェア
- 発行されたURL（例: `https://your-app.streamlit.app`）を共有
- これで友達はブラウザからすぐ使える！
